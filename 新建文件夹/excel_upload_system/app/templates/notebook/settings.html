如何获取豆包AI API密钥？
  1. 访问 豆包开发者平台
  2. 注册/登录您的账号
  3. 进入「API密钥管理」页面
  4. 点击「创建新密钥」按钮
  5. 复制生成的API密钥
  6. 将密钥粘贴到上方输入框中
注意：API密钥属于敏感信息，请不要分享给他人或在代码中硬编码。
系统当前使用豆包AI API，而非火山API。如果您想使用火山API，需要修改系统代码以支持火山引擎的API调用方式。
🔔 钉钉通知配置
配置说明: 钉钉机器人需要通过环境变量进行配置
如何配置钉钉机器人？
  1. 打开钉钉，创建一个群聊或选择现有群聊
  2. 点击群设置 → 智能群助手 → 添加机器人
  3. 选择「自定义」机器人
  4. 填写机器人名称，选择安全设置（推荐使用「自定义关键词」）
  5. 点击「完成」后，复制生成的Webhook URL
  6. 设置环境变量：
    • Windows: 在命令行中执行 set DINGTALK_WEBHOOK=您的webhook_url
    • Linux/Mac: 在终端执行 export DINGTALK_WEBHOOK=您的webhook_url
  7. 重启应用程序使配置生效
注意：请确保在安全设置中添加关键词，如「通知」、「测试」等，否则消息可能发送失败。
