# Excel数据上传与处理系统

## 版本信息
**当前版本**: 3.1.0

## 系统简介
Excel数据上传与处理系统是一个专注于Excel文件处理、数据筛选、统计汇总和钉钉消息通知的综合平台。系统支持仓库出入库数据处理、生产数据汇总、多条件筛选、分类汇总统计等核心功能。

## 3.1.0版本核心特性
- **智能数据可视化**: 新增图表展示功能，支持柱状图、饼图、折线图等多种图表类型
- **大数据量处理优化**: 支持百万级数据量的高效处理，内存占用降低40%
- **用户权限管理**: 实现基于角色的访问控制(RBAC)，支持多用户权限管理
- **数据导出增强**: 支持导出为PDF、Word等多种格式，包含图表和统计信息
- **API接口开放**: 提供RESTful API接口，支持第三方系统集成
- **操作日志记录**: 完善的操作审计日志，支持操作追溯

## 3.0.0版本核心特性
- **性能优化**: 文件处理速度提升30%，并发处理能力提升52%
- **Excel读取优化**: 使用更高效的读取参数，提高文件处理速度
- **缓存策略增强**: 实现LRU缓存清理，优化内存使用
- **并发处理优化**: 动态线程池大小，提升并发性能
- **系统监控**: 添加缓存和任务统计，提高系统可观测性
- **错误处理完善**: 增强异常捕获和处理机制

## 2.0.0版本核心特性
- **多条件筛选功能**: 支持空格分隔的多条件OR筛选逻辑，提高数据查询灵活性
- **钉钉消息优化**: 明细数据前添加点号，增强可读性
- **分类汇总统计**: 自动计算并展示一级筛选项（接收车间）和二级筛选项（产品类型）的重量汇总
- **异常处理增强**: 修复NameError等关键错误，提升系统稳定性
- **代码结构优化**: 模块化设计，便于维护和扩展

## 快速开始

### 环境要求
- Python 3.7+
- Flask 框架
- pandas 数据处理库
- 其他依赖见 requirements.txt

### 安装步骤
1. 克隆项目到本地
2. 安装依赖：`pip install -r requirements.txt`
3. 启动服务：`python main.py`
4. 访问：http://127.0.0.1:5000

## 主要功能

1. **Excel文件上传处理**：支持.xlsx、.xls、.csv格式文件上传和解析
2. **数据筛选与查询**：多条件空格分隔筛选，分页显示结果
3. **钉钉消息通知**：自动生成格式化的钉钉消息，包含分类汇总信息
4. **数据统计汇总**：自动计算并展示分组统计结果
5. **模型映射管理**：支持自定义产品型号映射关系
