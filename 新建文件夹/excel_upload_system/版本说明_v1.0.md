## 功能特性

### 核心功能
1. **CSV双层表头处理**
   - 自动识别双层表头
   - 灵活的列映射配置
   - 数据汇总功能

2. **工艺文件管理**
   - 文件上传和管理
   - 自定义字段管理
   - 数据导出功能
   - 钉钉分享功能

3. **模板管理**
   - 模板创建和编辑
   - 字段配置
   - 模板预览

4. **员工工装管理**
   - 员工信息管理
   - 工装分配记录
   - 工装状态追踪

5. **卷径计算工具**
   - 卷径自动计算
   - 参数配置
   - 结果展示

### 高级功能
6. **开红莲模式API**
   - 智能数据处理
   - AI辅助分析
   - 批处理功能

7. **钉钉机器人集成**
   - 消息推送
   - 数据分享
   - 通知提醒

8. **豆包AI笔记**
   - AI笔记功能
   - 任务管理
   - 智能分析

### 系统特性
- Redis缓存支持
- 异步任务处理
- 响应式设计
- 移动端适配
- 完善的日志系统
- 版本管理

## 项目结构
```
excel_upload_system/
├── app/
│   ├── models/          # 数据模型
│   ├── routes/          # 路由处理
│   ├── services/        # 业务服务
│   ├── static/          # 静态资源
│   ├── templates/       # 模板文件
│   └── utils/           # 工具函数
├── data/                # 数据存储
├── redis/               # Redis相关
├── requirements.txt     # 依赖包
└── main.py              # 主入口
```

## 启动方式

### Windows
```bash
start_windows.bat
```

### 直接启动
