## 后端配置

### 步骤7：创建数据库表
在 MyDB Browser 或命令行中执行以下 SQL：

```sql
CREATE TABLE IF NOT EXISTS mentorship (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    mentor_id INTEGER NOT NULL,
    apprentice_id INTEGER NOT NULL,
    relationship_type TEXT DEFAULT 'direct',
    start_date TEXT,
    end_date TEXT,
    status TEXT DEFAULT 'active',
    notes TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(mentor_id, apprentice_id)
);

CREATE INDEX IF NOT EXISTS idx_mentorship_mentor ON mentorship(mentor_id);
CREATE INDEX IF NOT EXISTS idx_mentorship_apprentice ON mentorship(apprentice_id);
CREATE INDEX IF NOT EXISTS idx_mentorship_status ON mentorship(status);
```

### 步骤8：在 app.py 中添加蓝图
在 app.py 的导入部分添加：
```python
from routes.mentorship import mentorship_bp
```

在蓝图注册部分添加：
```python
app.register_blueprint(mentorship_bp, url_prefix='/api/mentorship')
```
