# AlgerMusicPlayer 使用说明

## 项目简介

AlgerMusicPlayer 是一个功能强大的第三方音乐播放器，支持网易云音乐账号登录与同步、音乐推荐、高品质音乐播放、桌面歌词等功能。

## 当前状态

✅ **开发服务器已成功启动**
✅ **Electron应用窗口已创建**

开发服务器地址：http://localhost:5173/

## 窗口显示问题解决方法

如果您看不到运行按钮或应用窗口，请尝试以下方法：

1. **检查任务栏**：应用可能已经在任务栏中运行，点击任务栏图标即可显示窗口

2. **使用Alt+Tab**：按Alt+Tab键查看所有打开的窗口，尝试找到AlgerMusicPlayer窗口

3. **检查系统托盘**：应用可能最小化到系统托盘，右键点击托盘图标选择显示窗口

4. **刷新应用窗口**：
   - 关闭当前运行的开发服务器（Ctrl+C）
   - 在AlgerMusicPlayer目录下重新运行 `npm run dev`

## 如何使用

1. **启动应用**：
   ```bash
   cd d:\新建文件夹\AlgerMusicPlayer
   npm install  # 首次运行需要安装依赖
   npm run dev  # 开发模式启动
   ```

2. **访问Web版本**：如果桌面应用有问题，您还可以直接在浏览器中访问 http://localhost:5173/

3. **搜索音乐**：在搜索框中输入歌曲名称或歌手名称进行搜索

4. **播放控制**：使用底部播放控制栏控制音乐播放

## 常见问题

**Q: 窗口已经创建但看不到？**
A: 尝试使用Alt+Tab查看所有打开的窗口，或检查任务栏和系统托盘

**Q: 开发服务器端口被占用？**
A: 可以修改配置文件中的端口设置，或关闭占用5173端口的其他应用

**Q: 无法播放音乐？**
A: 确保本地网络连接正常，应用依赖的netease-cloud-music-api服务已经启动

## 快捷键

- **播放/暂停**：空格键
- **上一首**：Ctrl+←
- **下一首**：Ctrl+→
- **音量调节**：Ctrl+↑/↓

详细的快捷键配置可以在应用设置中查看和自定义。
